轻量级自选股管理工具：支持沪深主板/科创板/创业板/北交所全市场股票，多分组看板、实时行情（东方财富+新浪双数据源）、网格/列表双视图、拖拽排序与置顶、自定义列、智能搜索补全、价格隐藏与涨跌提醒，数据本地存储，隐私安全。

一款专为 A 股投资者打造的 Chrome 浏览器扩展，帮助您高效管理自选股，实时追踪行情动态。

【核心功能】

📊 自选股分组管理
- 支持创建多个自定义分组（最多 20 个），按行业、主题、策略等维度分类管理
- 内置"全部"默认分组，汇总所有自选股
- 一键将股票移动或复制到其他分组
- 每个分组独立管理，互不干扰

📈 实时行情看板
- 列表视图：经典表格布局，一目了然查看所有股票行情
- 网格视图：卡片式布局，适合快速浏览大量股票
- 实时刷新：弹窗打开时每 10 秒自动更新，后台每 30 秒刷新
- 手动刷新：底部状态栏刷新按钮，一键拉取最新行情，旋转动画+toast 反馈
- 更新时间显示：状态栏实时显示"刚刚更新 / N秒前 / N分钟前"
- 数据来源：东方财富 API（主）+ 新浪财经 API（备），确保数据稳定可靠

📋 自定义列设置
- 自由选择显示的数据列：现价、涨跌幅、涨跌额、成交量、成交额、最高、最低、今开、昨收、代码、自选时间等
- 拖拽排序，打造专属看板布局
- 设置面板支持返回和快捷操作

📌 置顶与排序
- 一键置顶重要股票，始终显示在最前面；取消置顶自动排到非置顶区末尾
- 支持手动拖拽排序，置顶区和非置顶区各自独立编号，互不干扰
- 多维度自动排序：涨跌幅、现价、成交额、自选时间等

🔔 桌面提醒
- 浏览器工具栏图标实时显示当前排序第一的自选股涨跌幅
- 红色表示上涨，绿色表示下跌（遵循 A 股惯例）
- 鼠标悬停插件图标，查看排序前 5 只股票的行情概览
- 图标与弹窗排序完全一致，切换排序方式后图标即时联动

【数据说明】
- 行情数据来源于东方财富和新浪财经的公开 API
- 双数据源容灾：东方财富（主）→ 新浪财经（备）→ 演示数据（兜底）
- 支持沪深主板、科创板、创业板、北交所全市场股票
- 所有数据仅供参考，不构成投资建议
- 当 API 不可用时，会显示标注"演示数据"的示例内容

【隐私保护】
- 不收集任何用户个人数据
- 所有配置仅存储在本地浏览器中
- 不使用远程代码，所有逻辑本地执行
- 详细隐私政策请参见插件详情页

【适用场景】
- 日常盯盘：快速查看自选股实时行情
- 分组管理：按行业、概念、策略分类管理股票池
- 涨跌监控：通过工具栏图标随时掌握排序第一只自选股动态
- 高效整理：自定义列和排序，打造专属看板

【快捷操作】
- 点击插件图标打开弹窗
- 输入股票代码或名称添加自选股
- 代码搜索支持拼音首字母、中文名称、行业标签
- 支持沪深所有 A 股 + 科创板 + 北交所
- 右键菜单快速操作（如适用）
