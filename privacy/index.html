股票提醒助手 - 隐私权政策

Stock Alert Assistant - Privacy Policy

最后更新日期：2026年7月10日

一、概述

「股票提醒助手」（以下简称"本插件"）是一款 Chrome 浏览器扩展，提供自选股分组与看板管理功能。本插件尊重并保护用户隐私，不收集、不存储、不传输任何用户个人数据。所有用户配置数据仅保存在用户本地浏览器中，不会发送到任何服务器。

核心承诺：本插件不收集任何个人身份信息、浏览记录、位置数据或其他用户隐私数据。插件仅访问公开的股票行情 API 以展示实时行情信息。

二、数据收集说明

本插件不收集以下任何类型的用户数据：

  • 个人身份信息（姓名、邮箱、电话等）
  • 健康信息
  • 财务和付款信息（不处理任何交易或支付数据）
  • 身份验证信息（密码、凭证等）
  • 个人通讯（邮件、短信、聊天记录）
  • 位置信息（GPS 或 IP 定位）
  • 网络浏览记录
  • 用户活动数据（点击、鼠标、键盘行为）
  • 网站内容（页面文字、图片、视频）

所有用户配置（自选股列表、分组、排序、列设置等）均通过 chrome.storage.local 存储在用户本地浏览器中，不会同步到云端，也不会发送给开发者或任何第三方。

三、权限说明

本插件请求的权限及其用途如下：

权限用途说明
storage用于在本地浏览器中存储用户的自选股列表、分组配置、排序偏好、列设置等数据，确保用户关闭浏览器后配置不丢失。数据仅存储在本地，不会同步到云端。
alarms用于设置定时器，定期刷新股票行情数据（每 30 秒），以实现实时行情展示和涨跌提醒功能。

四、主机权限说明

本插件需要访问以下外部 API 以获取公开的股票行情数据：

主机用途说明
push2.eastmoney.com东方财富实时行情 API（主要数据源），获取股票实时价格、涨跌幅等行情数据。
searchapi.eastmoney.com东方财富搜索建议 API，用于添加自选股时的股票代码/名称/拼音搜索补全。
hq.sinajs.cn新浪财经实时行情 API（备用数据源），在主数据源不可用时作为备选。
重要说明：插件仅从上述 API 获取公开的股票行情数据（价格、涨跌幅、成交量等），不会向这些服务发送任何用户数据。请求中不包含用户身份信息、浏览记录或其他个人数据。

五、数据存储

本插件的所有数据存储均在用户本地浏览器中完成：

  • 存储位置：Chrome 浏览器本地存储（chrome.storage.local）
  • 存储内容：自选股列表、分组配置、排序偏好、列设置等
  • 存储期限：数据保存在用户浏览器中，直到用户主动删除或卸载插件
  • 数据同步：不会将数据同步到云端或任何远程服务器

六、第三方服务

本插件使用以下第三方公开 API 获取股票行情数据：

  • 东方财富网（eastmoney.com）— 提供实时股票行情数据和搜索建议服务
  • 新浪财经（sina.com.cn）— 提供备用实时股票行情数据

这些服务仅提供公开的市场数据，本插件不会向其发送任何用户数据。用户使用本插件时，上述服务的使用条款和隐私政策同样适用。

七、数据安全

由于本插件不收集、不传输任何用户个人数据，因此不存在数据泄露风险。所有配置数据存储在用户本地浏览器中，受浏览器自身的安全机制保护。插件不使用任何远程代码，所有逻辑均在本地执行（CSP: script-src 'self'）。

八、儿童隐私

本插件不面向 13 岁以下儿童，也不会有意收集儿童的个人信息。由于本插件不收集任何用户数据，因此对儿童隐私无任何影响。

九、远程代码声明

本插件不使用任何远程代码。所有 JavaScript 代码均打包在插件内部，通过 Manifest V3 的内容安全策略（CSP: script-src 'self'; object-src 'self'）确保不会加载或执行任何外部脚本。

十、政策变更

如果本隐私权政策发生变更，更新后的政策将发布在此页面上，并更新页面顶部的"最后更新日期"。由于本插件不收集用户数据，用户无需担心政策变更对其隐私的影响。
